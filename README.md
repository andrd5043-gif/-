## About

TaskFlow позволяет создавать проекты, распределять задачи между участниками, отслеживать прогресс через Kanban-доску и получать уведомления о дедлайнах. Есть роли (админ, менеджер, исполнитель), комментарии к задачам, фильтры по статусу и приоритету, а также простая аналитика по загрузке команды. Идеально подходит для небольших IT-команд и студенческих проектов.
